# 🚀 Инструкция по запуску P2P Tracker

## Стек
- **Frontend:** React (готовый код)
- **База данных + авторизация:** Supabase (бесплатно)
- **Хостинг:** Vercel (бесплатно)
- **Итог:** рабочий сайт с логинами по типу `https://p2p-tracker-xxx.vercel.app`

---

## ШАГ 1 — Supabase (база данных)

1. Зайди на **https://supabase.com** → **Start your project** (бесплатный аккаунт)
2. Создай новый проект:
   - Name: `p2p-tracker`
   - Database Password: придумай и сохрани
   - Region: выбери ближайший (например, `Frankfurt`)
3. Подожди ~2 минуты пока проект создастся

### Создать таблицы:
4. В левом меню → **SQL Editor** → **New query**
5. Скопируй ВЕСЬ код из файла `supabase_schema.sql` и нажми **Run**
6. Должно появиться `Success. No rows returned`

### Получить ключи:
7. Левое меню → **Settings** → **API**
8. Скопируй:
   - `Project URL` → это `REACT_APP_SUPABASE_URL`
   - `anon public` key → это `REACT_APP_SUPABASE_ANON_KEY`

---

## ШАГ 2 — GitHub (загрузить код)

1. Зайди на **https://github.com** → создай аккаунт если нет
2. Создай новый репозиторий: **New repository** → name: `p2p-tracker` → **Public** → **Create**
3. Загрузи папку проекта:

```bash
# В терминале перейди в папку проекта
cd p2p-tracker

# Создай .env файл (НЕ загружай его в GitHub!)
cp .env.example .env
# Вставь свои ключи из Supabase в .env

# Инициализируй git и загрузи
git init
git add .
git commit -m "init"
git branch -M main
git remote add origin https://github.com/ТВО_ИМЯ/p2p-tracker.git
git push -u origin main
```

**Альтернатива без терминала:** на GitHub нажми **uploading an existing file** и перетащи все файлы вручную.

---

## ШАГ 3 — Vercel (хостинг)

1. Зайди на **https://vercel.com** → войди через GitHub
2. **Add New Project** → выбери репозиторий `p2p-tracker`
3. **Framework Preset:** выбери `Create React App`
4. **Environment Variables** — добавь оба ключа:
   ```
   REACT_APP_SUPABASE_URL = https://xxx.supabase.co
   REACT_APP_SUPABASE_ANON_KEY = eyJhbG...
   ```
5. Нажми **Deploy** → подожди ~2 минуты
6. Готово! Получишь ссылку вида `https://p2p-tracker-xxx.vercel.app`

---

## ШАГ 4 — Создать первого админа

1. Открой свой сайт → зарегистрируйся с Email и паролем
2. Вернись в Supabase → **SQL Editor** → выполни:
   ```sql
   UPDATE public.profiles SET role = 'admin' WHERE email = 'твой@email.com';
   ```
3. Перезайди на сайт — увидишь раздел **Адмін**

---

## ШАГ 5 — Добавить сотрудников

Через **Адмін-панель** на сайте:
- Введи имя, email и пароль для каждого сотрудника
- Выбери роль `worker`
- Отправь им ссылку и пароль

---

## Роли

| Роль | Что видит и может |
|------|------------------|
| **admin** | Все ордера всех сотрудников, статистика, пары/спред, адмін-панель, удаление |
| **worker** | Только свои ордера, добавление новых ордеров, просмотр пар |
